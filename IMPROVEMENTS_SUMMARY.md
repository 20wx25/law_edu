### 4. 修复具体内容问题

#### a) 删除不合理的"邻居听到"情节
**问题**: 原版中在里屋大声说"不用了！我要回家了！"能让邻居听到，但邻居在外面，不太合理。

**解决方案**:
- 删除了该选项
- 新增了更合理的分支 `act3A_firm_rejection`：
  ```
  + [推开他的手，大声说："我不舒服，我要回家！"]
      你的心跳加快，推开他的手。

      "我不舒服，我要回家！"

      你的坚决态度让孙伯伯愣了一下。

      -> act3A_firm_rejection
  ```
- 在 `act3A_try_leave` 中保留了大声呼救选项，但改为让孙伯伯因害怕而放行，而非邻居介入

#### b) 修改时间描述
**修改前**: "你跌跌撞撞地走出孙伯伯家，天已经完全黑了。"

**修改后**: "你从孙伯伯家离开的时候天已经完全黑了。"

**位置**: `act4_silence_path` (第360行)

#### c) 增加保留证据问题
在 `police_investigation` 场景中增加了教育性互动：

```
💡 你有保留证据吗？

+ [有，我当时的衣服包括内衣都原封不动地留着，也没有洗澡]
    女警察赞许地点头："你做得非常好！这些证据对案件调查很关键。"
    -> evidence_preserved_path

+ [我太害怕了，回家就洗澡换衣服了]
    女警察温柔地说："没关系，医院检查仍然可能发现证据。而且你的陈述和及时报警也很重要。"
    -> evidence_partial_path
```

新增两个结局分支：
- `evidence_preserved_path`: 保留了证据
- `evidence_partial_path`: 未能保留完整证据

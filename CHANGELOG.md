# 普法情景游戏 - 更新日志

## [v1.3] - 2026-03-30 用户修订版

### 修改内容
用户基于v1.2版本进行了以下修订：

#### 1. 简化提示语言
- **prologue**: 修改开场提示，更简洁直接
  - 改前: "💡 记住：遇到危险时，寻求帮助永远是正确的选择。"
  - 改后: "💡 提示：遇到让你不舒服的情况时，尽快离开现场，并告诉可信任的大人。"

#### 2. 删除章节编号
- 删除所有"第X幕"标题前缀
  - "第一幕：放学路上" → "放学路上"
  - "第二幕A：孙伯伯家" → "孙伯伯家"
  - "第三幕A：危险升级" → "危险升级"
  - "第四幕：沉默还是求助？" → "沉默还是求助？"

#### 3. 精简法律小课堂内容
多处法律小课堂内容被精简或重新组织：

**act3A_danger_path**:
- 改前: 详细的《未成年人保护法》第40条全文
- 改后: "💡 提示：当成年人反复评论你的外貌、靠得过近，或让你感到不舒服时，要提高警惕。让你不舒服的关注，不是你必须接受的'好意'。"

**act3A_stage2**:
- 改前: 详细的"防性侵安全四步法"
- 改后: 简化为三条要点

**act3A_try_leave**:
- 改前: 完整的《刑法》第238条条文
- 改后: 简化为核心要点

**tell_parents_attempt**:
- 重新组织了《未成年人保护法》第11条的展示方式
- 将求助途径列表整合到法律小课堂中

#### 4. 删除legal_education_finale场景
- 原版在evidence_preserved_path和evidence_partial_path后都会进入legal_education_finale
- 修改后直接进入ending_A2_assault_reported
- 删除了集中展示所有法律条文的场景

#### 5. 修改结局展示
**ending_A2_assault_reported**:
- 改前: 使用"【结局复盘】"格式
- 改后: 使用"【安全提示】"格式，更清晰的checklist

**ending_D_severe**:
- 改前: 使用"【结局复盘】"
- 改后: 使用"【深刻教训】"
- 删除了legal_education_finale跳转，改为直接END

#### 6. 调整提示语气
多处提示改为更温和、更建议性的语气：
- "大声呼救是保护自己的有效方式" → "可以尝试通过大声呼救来引起附近其他人的注意"
- "如果父母不相信你，老师、学校辅导员、警察都是可以求助的对象！" → 改为更委婉的表述

## 如何使用版本控制

### 查看历史版本
```bash
# 进入versions文件夹
cd versions/

# 查看特定版本
cat v1.2/普法情景游戏_v1.2.ink
```

### 恢复旧版本
```bash
# 恢复v1.2为当前工作版本
cp versions/v1.2/普法情景游戏_v1.2.ink 普法情景游戏.ink
```

### 对比版本差异
```bash
# 对比v1.2和v1.3的差异
diff versions/v1.2/普法情景游戏_v1.2.ink versions/v1.3/普法情景游戏_v1.3.ink
```

### 测试Web版本
```bash
# 在浏览器中打开v1.3的Web版本
open versions/v1.3/v1.3_web/index.html
```

## 注意事项

1. **不要修改versions文件夹**中的任何文件，这些是历史备份
2. **修改主文件**：只修改根目录的 `普法情景游戏.ink`
3. **创建新版本**：如有重大修改，应创建新的版本备份（如v1.4）
4. **Web版本**：只有v1.0和v1.3有Web版本，其他版本需要在Inky中导出

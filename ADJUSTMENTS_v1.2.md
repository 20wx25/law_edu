## ✅ 完成的调整

### 1. 删除重复选项 (act3A_stage2)

**解决方案**:
- ✅ 删除了 "推开他的手，大声说：'我不舒服，我要回家！'" 选项
- ✅ 删除了相关的 `act3A_firm_rejection` 分支
- ✅ 保留了 "躲开他的手，站起来说'我真的要走了'" 选项

**修改位置**: act3A_stage2 部分

### 2. 合并相同结果选项 (act2A_hall_stay)
**问题**: "说'我该回家了'，站起身准备离开" 和 "找个借口离开：'我忘了奶奶让我买东西'" 导向相同结果

**解决方案**:
- ✅ 删除了 "找个借口离开：'我忘了奶奶让我买东西'" 选项
- ✅ 保留了 "说'我该回家了'，站起身准备离开" 选项
- ✅ 简化了选择分支，避免冗余

**修改前**:
```
+ [说"我该回家了"，站起身准备离开]
    ...
+ [找个借口离开："我忘了奶奶让我买东西"]
    ...
```

**修改后**:
```
+ [说"我该回家了"，站起身准备离开]
    ...
```

### 3. 调整大声呼救提示文字 (act3A_loud_shout)
**问题**: 原文"不要担心'小题大做'——你的安全最重要"语气过于强硬

**解决方案**:
- ✅ 修改为更温和的建议性语气

**修改前**:
```
💡 大声呼救是保护自己的有效方式。不要担心"小题大做"——你的安全最重要。
```

**修改后**:
```
💡 可以尝试通过大声呼救来引起附近其他人的注意，是保护自己的有效方式。
```

**修改位置**: act3A_loud_shout 第347行

---

### 4. 改进离开场景描述 (act4_silence_path)
**问题**: 原文直接叙述"你从孙伯伯家离开的时候天已经完全黑了"后立即进入"回到家"，缺少过渡

**解决方案**:
- ✅ 将"你从孙伯伯家离开的时候天已经完全黑了"作为场景设置
- ✅ 添加可点击选项："你跌跌撞撞地往家走去"
- ✅ 新增 `act4_at_home` 场景，用于到家后的内容
- ✅ 增强了情感过渡和玩家参与感

**修改前**:
```
=== act4_silence_path ===
你从孙伯伯家离开的时候天已经完全黑了。

回到家，奶奶看到你脸色苍白...
```

**修改后**:
```
=== act4_silence_path ===
你从孙伯伯家离开的时候天已经完全黑了。

+ [你跌跌撞撞地往家走去]
    -> act4_at_home

=== act4_at_home ===
回到家，奶奶看到你脸色苍白...
```

**修改位置**: act4_silence_path 第322-327行

---

### 5. 自然化保留证据问题 (police_investigation)
**问题**: 原文使用 "💡 你有保留证据吗？" 作为玩家选择，不够自然

**解决方案**:
- ✅ 将提示改为女警察的对话："小朋友，回家后你有洗澡或者换衣服吗？"
- ✅ 调整选项文字，使其更符合回答问题的语气
- ✅ 保留了教育意义的同时增强了对话的真实感

**修改前**:
```
女警察还告诉你："小朋友，这不是你的错。你做得对，报警能让坏人受到惩罚。"

💡 你有保留证据吗？

+ [有，我当时的衣服包括内衣都原封不动地留着，也没有洗澡]
    女警察赞许地点头："你做得非常好！这些证据对案件调查很关键。"

+ [我太害怕了，回家就洗澡换衣服了]
    女警察温柔地说："没关系，医院检查仍然可能发现证据..."
```

**修改后**:
```
女警察还告诉你："小朋友，这不是你的错。你做得对，报警能让坏人受到惩罚。"

女警察接着问你："小朋友，回家后你有洗澡或者换衣服吗？"

+ [没有，我当时的衣服包括内衣都原封不动地留着，也没有洗澡]
    女警察赞许地点头："你做得非常好！这些证据对案件调查很关键。"

+ [我太害怕了，回家就洗澡换衣服了]
    女警察温柔地说："没关系，医院检查仍然可能发现证据..."
```

**修改位置**: police_investigation 第519-531行
